全て自己責任です。

  • １：renderのアカウントを作成(ログイン)

  • ２：blueprintをクリック

  • ３：New Blueprint Instanceをクリック

  • ４：Public Git repositoryの欄にGithubのURLをペースト、Continueを押す(GithubのURLは一番下にあります)

  • ５：適当な名前を付けてCreate New ResourcesもしくはApplyをクリック

  • ６：Dashboardに戻り作成したものをクリック、左上に表示されているURLが作成したインスタンスです


    • このインスタンスの最新版は環境変数(Environment Variables)でhcaptchaという名前の環境変数を作成してhCaptchaで取得したサイトキーをコピペして下さい
